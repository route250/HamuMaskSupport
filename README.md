# HamuMaskSupport

HamuMaskのサポートページを公開するためのリポジトリです。

## 公開ページ

- Google Chrome版: https://route250.github.io/HamuMaskSupport/googlechrome/
- Google Chrome版プライバシーポリシー: https://route250.github.io/HamuMaskSupport/googlechrome/privacy/

## 構成

- `googlechrome/`: はむマスク for Google Chrome のサポートページ
- `googlechrome/privacy/`: はむマスク for Google Chrome のプライバシーポリシー
